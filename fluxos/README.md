# Fluxos de Atendimento MCMV

## 📞 Atendimento Inicial
1. Cliente entra em contato
2. Solicitar dados:
   - Nome
   - CPF
   - Cidade
3. Verificar elegibilidade
4. Direcionar para próximo passo

---

## 📄 Análise de Cadastro
1. Conferir documentos
2. Validar renda
3. Verificar restrições
4. Aprovar ou reprovar

---

## 🏠 Pós-aprovação
1. Orientar cliente
2. Enviar instruções
3. Acompanhar processo
