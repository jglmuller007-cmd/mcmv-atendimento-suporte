# Extensões MCMV

Extensões para otimizar o atendimento e processos internos.

## Objetivo
- Automatizar tarefas
- Melhorar produtividade
- Reduzir erros

## Ideias de Extensões

- Preenchimento automático de dados do cliente
- Validação de CPF automática
- Copiar respostas padrão com um clique
- Identificação de erros em formulários

# Bugs encontrados

## Bug 1
Erro ao preencher CPF automaticamente

### Causa
Campo não identificado corretamente

### Solução
Ajuste no seletor do elemento
